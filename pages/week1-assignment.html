📝 Week 1 Assignment: Apply Your R Skills

Real-World Data Analysis with Crime Statistics

🎯 Assignment Overview

Put your new R programming skills to the test with a real-world data analysis assignment! You'll analyze Los Angeles crime data using the statistical techniques you learned in the tutorial and class activity.

📊

Crime Statistics Analysis

Analyze Los Angeles crime data from September 1, 2023, using frequency tables and histograms

👥

Gender-Based Comparison

Compare crime patterns between male and female victims using data filtering

📈

Data Visualization

Create and interpret histograms with different bin sizes to understand data patterns

✍️

Statistical Interpretation

Answer analytical questions about age distributions and data trends

📋 What You'll Do

Assignment Tasks:

1

Load and Explore Dataset

Load the LA Crime dataset and examine its structure using str() function

2

Create Frequency Tables

Generate frequency tables for victim ages divided into groups using table() and cut()

3

Generate Histograms

Create histograms to visualize age distributions with 10 bins using hist()

4

Filter Data by Gender

Separate data into male and female victim subsets using filter() function

5

Compare Visualizations

Create histograms for both gender groups and compare with different bin sizes (10 vs 20)

6

Answer Questions

Provide written analysis of data patterns and interpretation of results

📊 Dataset Information

LA Crime Dataset

You'll work with crime statistics from Los Angeles on September 1, 2023. The original dataset contains over 800,000 entries, but has been filtered to focus on one day with clear gender identification.

📁 File Name

LA_Data.csv

📏 Size

555 rows (observations)

📊 Columns

Vict.Age - Age of victim
Vict.Sex - Gender (F/M)

🌐 Source

Data.gov Crime Database

❓ Questions You'll Answer

❓

Data Structure Questions

  • What type of data is "Vict.Age"? How many observations does it have?
  • What type of data is "Vict.Sex"? What are the unique values?
❓

Data Analysis Questions

  • What do the frequency tables tell you about victim ages?
  • At what age range is someone most commonly a victim of crime?
❓

Comparative Analysis Questions

  • How do trends change when data is separated by gender?
  • What new inferences can you make from gender-separated data?
  • How does changing bin sizes affect data interpretation?
