Class Activity: Agricultural Data Exploration

Activity 1.1: Crop Yield Data

In this activity, you will explore a dataset containing crop yield information from different farms.

Instructions:

  1. Load the provided crop yield dataset
  2. Examine the structure of the data using str()
  3. Calculate basic summary statistics
  4. Identify different types of variables (quantitative vs. qualitative)

Sample Code:

# Load data
crop_data <- read.csv("crop_yields.csv")

# Examine structure
str(crop_data)
head(crop_data)

# Summary statistics
summary(crop_data)

# Variable types
class(crop_data$yield)
class(crop_data$farm_type)
