🧮 R as Your Agricultural Calculator

Learn to use R for practical farm calculations:

Field Area Calculation

# Calculate field area in hectares
length <- 100  # meters
width <- 50    # meters
area_sqm <- length * width
area_hectares <- area_sqm / 10000

Seed Requirements

# Calculate seeds needed
seeds_per_sqm <- 200
total_seeds <- area_sqm * seeds_per_sqm
seed_weight_kg <- (total_seeds / 1000) * 0.05
