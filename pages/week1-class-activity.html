💡 Activity Tips

🎯 Follow the Flow

Work through the notebook cells in order - later cells depend on earlier ones

🎮 Experiment

Try modifying the code examples to see what happens - that's how you learn!

❓ Use Help

Type ?function_name in any code cell to get help documentation

💾 Save Often

Download your work regularly - Binder environments are temporary

🔍 Look for Hints

Each exercise has helpful comments and hints to guide you

🎯 Complete "Your Turn" Sections

Look for emoji markers (🎯) indicating interactive exercises

⌨️ Keyboard Shortcuts

Master these shortcuts to code more efficiently:

Shift + Enter Run current cell and move to next
Ctrl + Enter Run current cell and stay in place
A Insert cell above
B Insert cell below
DD Delete current cell
M Change cell to Markdown

🆘 Need Help?

🔧 Technical Issues

  • Binder won't load? Try refreshing the page or clearing browser cache
  • Code not working? Check for typos and make sure you've run all previous cells
  • Lost your work? Always download files before closing Binder
