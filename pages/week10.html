ğŸ§® Key R Functions This Week

ANOVA Analysis

model <- lm(response ~ factor, data) # Fit ANOVA model
anova(model) # ANOVA table
summary(model) # Detailed output
mean(data$variable) # Calculate means

Linear Regression

model <- lm(y ~ x, data) # Simple regression
model <- lm(y ~ x1 + x2 + factor, data) # Multiple regression
summary(model) # Regression output
cor(x, y) # Correlation coefficient

Manual ANOVA Calculations

sum((group_mean - overall_mean)^2 * n) # Sum of squares between
sum((data - overall_mean)^2) # Total sum of squares
SSE <- SST - SSB # Sum of squares error
F_value <- MSB / MSE # F-statistic

ğŸ”§ Common Issues

  • ANOVA vs t-test? Use ANOVA when comparing 3+ groups
  • Sum of squares confusion? SST = SSB + SSE always
  • F-statistic interpretation? Large F suggests group differences
  • Regression assumptions? Check linearity, normality, equal variance

ğŸ“š Additional Resources
