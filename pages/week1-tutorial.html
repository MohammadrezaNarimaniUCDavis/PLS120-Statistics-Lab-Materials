ğŸ”¢ What You'll Learn: R Data Types

In R, data can be stored in different types, depending on the kind of information it represents. The basic data types in R are:

1. Integer

Whole numbers without decimals. Such as 1, 355, etc. Numbers like 2.5, 34.7 are NOT integers.

count <- 10L

2. Numeric

Numbers (both integers and decimals).

height <- 175.5

3. Character

Text or string data.

crop <- "wheat"

4. Logical

TRUE or FALSE values.

is_organic <- TRUE

5. Factor

Categorical data with levels.

treatment <- factor("control")

We can find out the data type of a variable by using the class() function. The output of this function is one word, specifying the data type or class.

Note: In R, the assignment operator <- is equivalent to = (like equal sign in math)

ğŸ“ˆ Data Visualization

Whenever looking at data, a good first step is to take a big picture approach, by looking at the distribution of the data points, and through simple visualization. It's often easier to look at a picture than it is to look at a bunch of numbers.

Frequency Tables

A frequency table shows the distribution of data given two values. This can also be described as looking at the "counts" of the data point.

# Simple frequency table
table(iris$Species)

# Two-way frequency table with grouping
frequency_table <- table(iris$Species, cut(iris$Sepal.Length, seq(4, 8, 0.5)))
frequency_table

Histograms

A histogram shows the range of data, location with the highest concentration of measurements, and shape of distribution (symmetric or skewed).

# Create a histogram
hist(iris$Sepal.Length,
     main = "Distribution of Sepal Length",
     xlab = "Sepal Length (cm)",
     col = "lightblue")

# Adjust number of bins
hist(iris$Sepal.Length, breaks = 15)  # More detail
hist(iris$Sepal.Length, breaks = 5)   # Less detail
